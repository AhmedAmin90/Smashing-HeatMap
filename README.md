# Creating Heat-Map

This assignment consists of heat map for Smashing Magazine.
In this part of the project we used:

- HTML5
- HTML5 semantic
- CSS3
- flex box
- css grid
- styleling

In this project we made sure to consider to :

- Align elements with flex and grid.
- The layout's width 1440px.
- Used article tag for articles.
